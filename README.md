# MNIST From Scratch

Classification des chiffres MNIST avec une régression logistique multi-classe implémentée from scratch (NumPy uniquement).

## Modèle
- Softmax + Cross-Entropy Loss
- Gradient Descent

## Structure
- `data/` : dataset mnist.csv
- `notebooks/` : notebook principal
- `src/` : modules Python
- `outputs/` : plots et résultats
